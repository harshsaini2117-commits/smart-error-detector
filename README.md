# Smart Error Detector

A Python-based tool that analyzes logs and identifies the most frequent error along with possible fixes.

## Features
- Extracts error messages from logs
- Identifies most frequent error
- Suggests possible fixes

## Tech Stack
- Python

## How to Run
1. Run: python error_detector.py
2. Paste logs
3. Type END
4. View results

## Future Improvements
- Machine learning-based suggestions
